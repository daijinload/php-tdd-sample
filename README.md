# PHP TDD SAMPLE

## require composer
[PHP7をwindows7にインストールする](http://qiita.com/daijinload/items/3eb3d7d776143b3c5392)

## setup

```bash
php composer.phar install
```

## run

```bash
./vendor/bin/peridot --force-colors tests/
```

## run and watch

```bash
./vendor/bin/peridot --force-colors --watch tests/
```
